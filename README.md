# 📊 Análise de Dados das Lojas - Challenge Data Science Oracle + Alura

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do Challenge de Data Science da Alura, com o objetivo de analisar dados de vendas de quatro lojas fictícias e extrair insights que auxiliem na tomada de decisão do negócio.

A análise foi realizada utilizando Python e a biblioteca Pandas, explorando informações como produtos vendidos, categorias, faturamento, frete, avaliações dos clientes e localização das compras.

---

## 🎯 Objetivos

- Importar e explorar os dados das quatro lojas.
- Realizar uma análise exploratória dos dados (EDA).
- Identificar padrões de vendas e comportamento dos clientes.
- Comparar o desempenho entre as lojas.
- Gerar insights que possam apoiar decisões estratégicas.

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Google Colab
- Git e GitHub

---

## 📂 Base de Dados

Os dados utilizados foram disponibilizados pela Alura através de arquivos CSV contendo informações de:

- Produto
- Categoria do Produto
- Preço
- Frete
- Data da Compra
- Vendedor
- Local da Compra
- Avaliação da Compra
- Tipo de Pagamento
- Quantidade de Parcelas
- Latitude e Longitude

Cada loja possui aproximadamente 2.359 registros de vendas.

---

## 📈 Análises Realizadas

### 🔍 Exploração dos Dados
- Visualização das primeiras linhas dos datasets.
- Verificação da estrutura dos dados.
- Identificação dos tipos de variáveis.
- Estatísticas descritivas.

### 🛒 Produtos e Categorias
- Quantidade de vendas por categoria.
- Produtos mais comercializados.
- Distribuição das categorias de produtos.

### 💰 Indicadores Financeiros
- Análise dos preços dos produtos.
- Avaliação dos custos de frete.
- Comparação de faturamento entre lojas.

### ⭐ Satisfação dos Clientes
- Distribuição das avaliações.
- Comparação do desempenho das lojas com base no feedback dos clientes.

### 📍 Análise Geográfica
- Distribuição das compras por estado.
- Avaliação do alcance regional das vendas.
